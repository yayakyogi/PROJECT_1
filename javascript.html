Pengertian Javascript

Javascipt adalah bahasa pemrograman komputer yang dinamis. Pada umumnya Javascipt digunakan pada web browser untuk menciptakan halaman web yang menarik, interaktif serta merapkan berbagai fungsi pada halaman web. Javascript adalah salah satu pemrograman web yang harus kita pelajari selain HTML dan CSS.

Kegunaan Javascript

Kegunaan utama JavaScript adalah untuk menuliskan fungsi yang disisipkan ke dalam HTML, baik disisipkan secara langsung maupun diletakan ke dalam file teks dan di link dari dokumen HTML. Secara fungsional, JavaScript dipakai untuk menyediakan akses script pada objek yang dibenamkan (embedded) Contoh penggunaan JavaScript, yaitu merubah image kursor ketika melewati objek tertentu, fungsi validasi pada form sebelum data dikirimkan ke server, dan sebagainya.
